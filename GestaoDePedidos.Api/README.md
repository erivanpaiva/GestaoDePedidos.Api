# Gestão de Pedidos

Mini aplicação web para gerenciamento de pedidos desenvolvida em C# com ASP.NET Core Web API e JavaScript.

## Overview

- Listagem de produtos
- Busca por nome do produto
- Filtro por categorias
- Checagem de ativos
- Carrinho de compras
- Pedidos com cálculo de total
- Registro de pagamentos
- Atualização de status de pedidos
- Listagem de pedidos detalhada

## Tecnologias Utilizadas

- C#
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- JavaScript
- HTML e CSS
